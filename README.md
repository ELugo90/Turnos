# Sistema de Turnos con QR

Sistema visual y corporativo para asignación y confirmación de turnos en eventos internos.

## Archivos
- asignar.html → Genera turno y QR
- monitor.html → Muestra turnos
- confirmar.html → Confirma acceso

## Nota
Este sistema usa localStorage.
Para múltiples dispositivos reales se recomienda backend.
